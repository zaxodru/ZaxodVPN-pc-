ZaxodVPN — это легковесная обходка для ПК (работает по принципу zapret), которая точечно подменяет параметры сетевых пакетов, чтобы восстановить доступ к заблокированным ресурсам. Никаких лишних окон, треев и регистраций. Просто запустил — и оно работает.

⚠️ Важно: Это не полноценный VPN. Это инструмент для глубокого анализа трафика (DPI) и его обфускации. Работает только на Windows.

✨ Особенности
Точечное воздействие: Правит только те соединения, которые попадают под блокировку (DPI).

Минимализм: Нет графического интерфейса. Только консоль и тишина.

WinDivert: Использует драйвер WinDivert для перехвата и модификации пакетов на лету.

Список стратегий: Поддерживает несколько режимов обхода (фрагментация, изменение TTL, подмена TCP-опций).

Для геймеров: Не режет пинг так, как это делает обычный VPN, так как не гоняет трафик через удаленный сервер.
