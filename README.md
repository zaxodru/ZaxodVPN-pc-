ZaxodVPN — это легковесная обходка для ПК (работает по принципу zapret), которая точечно подменяет параметры сетевых пакетов, чтобы восстановить доступ к заблокированным ресурсам. Никаких лишних окон, треев и регистраций. Просто запустил — и оно работает.

⚠️ Важно: Это не полноценный VPN. Это инструмент для глубокого анализа трафика (DPI) и его обфускации. Работает только на Windows.

✨ Особенности
Точечное воздействие: Правит только те соединения, которые попадают под блокировку (DPI).

Минимализм: Нет графического интерфейса. Только консоль и тишина.

WinDivert: Использует драйвер WinDivert для перехвата и модификации пакетов на лету.

Список стратегий: Поддерживает несколько режимов обхода (фрагментация, изменение TTL, подмена TCP-опций).

Для геймеров: Не режет пинг так, как это делает обычный VPN, так как не гоняет трафик через удаленный сервер.

🚀 Быстрый старт
Системные требования
Windows 7 / 8 / 10 / 11 (x64)

Права Администратора (обязательно для установки драйвера)

Установка и запуск
Скачай последний релиз (файл zaxodvpn.zip) со страницы Releases.

Распакуй архив в любую папку (например, C:\ZaxodVPN). Убедись, что путь не содержит кириллицы.

Запусти командную строку от имени Администратора.

Перейди в папку с программой:

cmd
cd C:\ZaxodVPN
Запусти обходку в автоматическом режиме:

cmd
zaxodvpn.exe --auto
Готово! Если всё прошло успешно, заблокированные сайты должны открываться в браузере.

⚙️ Использование
ZaxodVPN управляется через аргументы командной строки.

Основные команды
Автоматический режим (подбор стратегии):

cmd
zaxodvpn.exe --auto
Ручной режим (стандартная фрагментация, как в zapret):

cmd
zaxodvpn.exe --wfrag
Режим подмены TTL (если блокируют по времени жизни пакета):

cmd
zaxodvpn.exe --ttl 65
Остановка (завершение работы и выгрузка драйвера):
Нажми Ctrl + C в окне программы.

Расширенные настройки
Если нужен специфический порт или хост, используй комбинацию флагов:

cmd
zaxodvpn.exe --host 185.178.45.12 --port 443 --mode fragment
🐛 Отлавливаемые ошибки и их решения
Ошибка	Решение
Failed to open driver	Запусти консоль (CMD/PowerShell) от имени Администратора.
Address already in use	Закрой другие обходки (Zapret, GoodbyeDPI), они конфликтуют.
WinDivert not found	Антивирус удалил или заблокировал WinDivert.dll. Добавь папку в исключения антивируса и распакуй архив заново.
Сайты всё ещё не открываются	Попробуй сменить стратегию: zaxodvpn.exe --auto (или перезагрузи компьютер для сброса сетевых кэшей).
🛠 Сборка из исходников
Если хочешь собрать сам (при наличии исходного кода):

Установи Visual Studio 2022 с компонентом "Разработка классических приложений на C++".

Склонируй репозиторий:

cmd
git clone https://github.com/tvoe/username/zaxodvpn.git
Открой решение .sln и собери конфигурацию Release для x64.
