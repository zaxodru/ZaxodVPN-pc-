📦 Установка
Скачайте архив или папку с файлами. Расспакуйте из архива в папку и в удобное место. В папке (далее по инструкции):

**Основные функции:**

- **1. Install Service** – установка выбранного пресета как службы Windows (автозапуск при старте системы).
- **2. Remove Services** – удаление всех установленных служб zapret и WinDivert.
- **3. Check Status** – проверка состояния служб и процесса `winws.exe`.

**Настройки:**

- **4. Game Filter** – включает/отключает фильтр для игрового трафика (широкий диапазон портов). Полезно для онлайн-игр.
- **5. IPSet Filter** – переключение режимов работы со списками IP-адресов (any/none/loaded).
- **6. Auto-Update Check** – автоматическая проверка обновлений при запуске.

**Обновления:**

- **7. Update IPSet List** – загружает актуальный список IP-адресов для обхода.
- **8. Update Hosts File** – проверяет и обновляет файл `hosts` (если требуется).
- **9. Check for Updates** – проверяет наличие новой версии ZaxodVPN.

**Инструменты:**

- **10. Run Diagnostics** – запускает диагностику системы (проверка конфликтов, служб, DNS и т.д.).
- **11. Run Tests** – запускает тесты конфигурации (требуется PowerShell).

### Пресеты (готовые стратегии)

В папке `presets\` находятся файлы с готовыми параметрами запуска `winws.exe`:

- `ZaxodVPN - General.bat` – универсальная стратегия.
- `ZaxodVPN - Alt.bat` – альтернативная стратегия.
- `ZaxodVPN - Fake TLS.bat` – имитация TLS-рукопожатия.
- `ZaxodVPN - QUIC.bat` – обход с акцентом на протокол QUIC.

Вы можете запускать их напрямую (от администратора) для быстрого включения обхода без установки службы.

## 🌐 Важно: настройка DNS

Для корректной работы обхода необходимо использовать **безопасные DNS-серверы**, поддерживающие шифрование (DNS-over-HTTPS/TLS). Иначе провайдер сможет подменить ответы DNS и заблокировать доступ даже при работающем zapret.

**Рекомендуемые DNS:**
- `1.1.1.1` (Cloudflare)
- `8.8.8.8` (Google)

**Как настроить в Windows 10/11:**
1. Откройте «Параметры» → «Сеть и Интернет» → «Свойства адаптера».
2. Правый клик по вашему подключению (Wi-Fi или Ethernet) → «Свойства».
3. Выберите «IP версии 4 (TCP/IPv4)» → «Свойства».
4. Включите «Использовать следующие адреса DNS-серверов» и введите:
   - Предпочитаемый: `1.1.1.1`
   - Альтернативный: `8.8.8.8`
5. Нажмите OK.

Также можно настроить зашифрованный DNS через браузер или в параметрах Windows 11.

## ⚠️ Возможные проблемы и их решение

### 1. Антивирус блокирует `winws.exe` или `WinDivert64.sys`
- Добавьте папку `ZaxodVPN` в исключения антивируса.
- Эти файлы являются легитимными инструментами для работы с сетью и не содержат вредоносного кода.

### 2. Не открываются сайты после запуска
- Попробуйте другой пресет (например, `ZaxodVPN - Alt.bat`).
- Проверьте настройки DNS (см. выше).
- Запустите диагностику (пункт 10 в меню) и посмотрите, есть ли конфликты с другими службами (например, GoodbyeDPI, Killer, Check Point).

### 3. Ошибка «WinDivert service already in use»
- Убедитесь, что другие обходчики (GoodbyeDPI, старые версии zapret) остановлены и удалены.
- Воспользуйтесь пунктом меню **2. Remove Services**, затем перезагрузите компьютер и попробуйте снова.

### 4. Программа не запускается от имени администратора
- Правый клик по `service.bat` → «Запуск от имени администратора». Или создайте ярлык и в его свойствах укажите запуск от администратора.

### 5. После обновления Windows перестал работать
- Запустите диагностику (пункт 10) — она может выявить конфликты.
- Попробуйте переустановить службу (сначала удалите, потом установите заново).
